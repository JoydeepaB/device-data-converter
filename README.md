# Device Data Converter

## Overview

This project was completed as part of a Deloitte Technology Virtual Experience Program.

The objective was to transform device data received in multiple JSON formats into a standardized structure that could be processed consistently across systems. The solution automatically identifies the input format, converts it into a common schema, and validates the output using automated unit tests.

Through this task, I gained practical experience in Python programming, JSON data processing, data transformation, and software testing.
